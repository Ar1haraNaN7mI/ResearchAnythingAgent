# Diagram exports (Next AI Draw.io)

PNG, SVG, and `.drawio` files saved by MCP `export_diagram` or copied here manually.

- Default directory when `DRAWIO_EXPORT_DIR` is unset: this folder (`drawio_exports/` at repo root).
- PNG background default: **transparent**; switch with chat `drawio bg white` or env `DRAWIO_EXPORT_PNG_BACKGROUND`.

Binary exports are gitignored; this README is tracked.
